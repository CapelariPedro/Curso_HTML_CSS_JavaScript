## **1. Criando uma Página Pessoal com Estrutura HTML e Estilos CSS Avançados**

### **Objetivo**
Criar uma **página pessoal** utilizando **HTML e CSS** com diferentes seções bem estruturadas e estilizadas.

### **Requisitos Obrigatórios**
- **Cabeçalho (`<header>`)** contendo seu nome e um menu de navegação com links para as seções da página.
- **Seção "Sobre Mim" (`<section>`)** com uma imagem (`<img>`) e um parágrafo (`<p>`) descrevendo você.
- **Seção "Habilidades" (`<section>`)** com uma **lista (`<ul>` ou `<ol>`)** de pelo menos 5 habilidades.
- **Seção "Contato" (`<section>`)** contendo um **formulário (`<form>`)** com os seguintes campos:
  - Nome (`<input type="text">`)
  - Email (`<input type="email">`)
  - Mensagem (`<textarea>`)
  - Botão de envio (`<button>`)
- **Rodapé (`<footer>`)** com um link para suas redes sociais.

### **Requisitos de CSS**
- O **fundo da página** deve ter uma **cor degradê (gradient)**.
- O menu de navegação deve ter **links estilizados** que mudam de cor ao passar o mouse.
- A imagem da seção "Sobre Mim" deve ter **bordas arredondadas (`border-radius`)**.
- O formulário deve ser **centralizado** e ter um **efeito de sombra (`box-shadow`)**.

## **2. Construindo um Blog Simples com HTML e CSS**

### **Objetivo**
Criar um **blog básico** com **três postagens**, estruturado e estilizado com HTML e CSS.

### **Requisitos Obrigatórios**
- **Cabeçalho (`<header>`)** com o nome do blog e um menu de navegação.
- **Seção de Postagens (`<main>`)** contendo três postagens (`<article>`), cada uma com:
  - **Um título (`<h2>`)**
  - **Uma imagem (`<img>`)** relacionada ao tema
  - **Um parágrafo (`<p>`)** com um resumo do artigo
  - **Um botão "Leia Mais" (`<button>`)** estilizado com CSS
- **Rodapé (`<footer>`)** com links para redes sociais e um aviso de "Direitos Reservados".

### **Requisitos de CSS**
- O layout das postagens deve ser organizado em **cards estilizados** com **bordas arredondadas e sombra (`box-shadow`)**.
- Os botões "Leia Mais" devem ter um **efeito de hover** (mudar de cor ao passar o mouse).
- O texto dos artigos deve ter **um espaçamento agradável (`line-height`)** e uma fonte bem legível.
- A página deve se adaptar bem para **diferentes tamanhos de tela**.


---

## **Instruções**
1. Crie um arquivo `index.html` e adicione a estrutura base do HTML.
2. Crie um arquivo `style.css` para os estilos da página.
3. Adicione os elementos HTML conforme os requisitos.
4. Estilize a página aplicando os conceitos de CSS.
5. Teste em diferentes tamanhos de tela e refine a responsividade.
6. Caso deseje, implemente os desafios extras para aprimorar o projeto.

**Boa prática!**
